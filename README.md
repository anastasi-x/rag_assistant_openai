# RAG Assistant OpenAI

Готовый RAG-ассистент на базе OpenAI API.

Проект реализует RAG-пайплайн: загружает текстовые документы, разбивает их на чанки, создает embeddings, сохраняет данные в ChromaDB, ищет релевантный контекст по запросу пользователя и передает найденные фрагменты в OpenAI-модель для генерации ответа.

## Что умеет ассистент

* отвечает на вопросы по базе знаний;
* использует OpenAI API для генерации ответов;
* использует ChromaDB как векторное хранилище;
* применяет `top_k=3` для выбора релевантных чанков;
* сохраняет повторяющиеся ответы в SQLite-кэш;
* показывает статистику по базе знаний и кэшу.

## Установка и запуск

1. Установить Python 3.11.

2. Создать и активировать виртуальное окружение.

3. Установить зависимости:

```bash
pip install -r requirements.txt
```

4. Создать файл `.env` по примеру `.env.example`:

```env
OPENAI_API_KEY=your_openai_api_key_here
```

5. Перейти в папку ассистента:

```bash
cd assistant_api
```

6. Запустить приложение:

```bash
python app.py
```

## Команды в консоли

* обычный вопрос — ассистент ищет ответ в базе знаний;
* `stats` — показывает статистику системы;
* `clear` — очищает кэш;
* `quit`, `exit` или `q` — завершает работу.

## Используемые технологии

* Python 3.11
* OpenAI API
* ChromaDB
* SQLite
* RAG Pipeline

Дополнительно в проекте есть файл `evaluate_ragas.py` для оценки качества ответов через RAGAS.
